# 原型模式

这个设计模式就会想到一个克隆，提起克隆就会想到浅克隆和深克隆。

原型模式就是创建一个具有相同成员变量，相同成员方法的类。

## 通常的方式

- bean

```java
package com.designpatten.yuanxing.chuantong.bean;

public class People {
    private String name;
    private int age;

    public People(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    @Override
    public String toString() {
        return "People{" +
                "name='" + name + '\'' +
                ", age=" + age +
                '}';
    }
}

```

## 浅克隆

- beanOne

```java
package com.designpatten.yuanxing.yuanxing.bean;

public class MyString {
    private String str;
    public MyString(String str){
        this.str = str;
    }

    public String getStr() {
        return str;
    }

    public void setStr(String str) {
        this.str = str;
    }

    @Override
    public String toString() {
        return "MyString{" +
                "str='" + str + '\'' +
                '}';
    }
}

```

- beanTwo

```java
//如果需要该类 可以实现克隆的，就需要实现这个接口，然后实现相应的方法
public class People implements Cloneable{
    private String name;
    private int age;
    private MyString string;

    public MyString getString() {
        return string;
    }

    public void setString(MyString string) {
        this.string = string;
    }

    public People(String name, int age, MyString string) {
        this.name = name;
        this.string = string;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    @Override
    public String toString() {
        return "People{" +
                "name='" + name + '\'' +
                ", age=" + age +
                ", string=" + string +
                '}';
    }

    @Override
    public People clone() throws CloneNotSupportedException {
        People people = null;
        try {
            people = (People) super.clone();
        }catch (Exception e){}
        return people;
    }
}

```



- AppMain

## 其他

拷贝的是不会执行构造函数，它的执行是在堆中直接进行的，new的时候，需要进行一次类的加载，初始化等步骤，类加载过程中会执行构造函数，最后的对象放在堆中。克隆是直接复制堆中的二进制对象，然后重新分配一块内存块。
